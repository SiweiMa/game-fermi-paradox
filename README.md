# 地球探险（Earth Ascent）

一个 2D 纵向攀登类的极简科幻点击游戏原型：你操控的不是“人”，而是「人类的探索意志」。

## 玩法（MVP）

- 星球在宇宙中纵向分布，点击“高亮可达”的星球进行探索
- 只有一个核心能力值：探索范围 `X`
- 抵达星球会获得激励 `H`，并使 `X = X + H`
- 停留未行动时，`X` 会随时间衰减
- 当没有任何未探索且距离 `<= X` 的星球时，游戏结束
- 抵达 `New Earth` 立即通关

## 运行

直接用浏览器打开 `index.html` 即可（推荐 Chrome / Edge / Safari）。

如果你想用本地静态服务器：

```bash
python3 -m http.server 5173
```

然后访问 `http://localhost:5173`。
